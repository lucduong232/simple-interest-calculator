# Simple Interest Calculator

This project is a simple interest calculator.

## Description
It calculates simple interest based on:
- Principal amount
- Rate of interest
- Time period

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Example
If:
- Principal = 1000
- Rate = 5%
- Time = 2 years

Then:
Simple Interest = (1000 × 5 × 2) / 100 = 100
